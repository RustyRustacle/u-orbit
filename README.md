## 🌌 Introduction

**U-ORBIT** is a premier student-led Web3 ecosystem and DAO-like initiative based at Universitas Tarumanagara (UNTAR). We are dedicated to bridging the gap between academic theory and the real-world blockchain industry. 

Our mission is to cultivate a new generation of **ASTROs** (Agile, Skilled, Tenacious, Resilient, Optimistic) — elite builders and thinkers ready to take on the global Web3 ecosystem.

---

## 🚀 The Pipeline

Our framework is designed to continuously accelerate talent through three core phases:

1. **🧠 Learn**
   Foundational and advanced education in blockchain fundamentals, smart contract development, tokenomics, and decentralized governance. We host workshops, study groups, and deep-dive seminars.
2. **🏗️ Build**
   Hands-on experience. Members are organized into pods to architect and develop decentralized applications (dApps), protocols, and tooling. We emphasize shipping real products.
3. **💰 Earn**
   Connecting our top talent with global bounties, hackathons, grants, and direct employment opportunities within the Web3 space. We help you monetize your skills.

## 💻 Tech Stack

This repository houses the main landing page and platform for U-ORBIT, engineered for maximum performance, aesthetic fidelity, and developer experience.

- **Framework**: [Next.js (App Router)](https://nextjs.org/) for server-side rendering and static site generation.
- **Styling**: [Tailwind CSS (v4)](https://tailwindcss.com/) for utility-first, scalable design.
- **Animation**: [Framer Motion](https://www.framer.com/motion/) for fluid, Web3-native micro-interactions and transitions.
- **Language**: [TypeScript](https://www.typescriptlang.org/) for end-to-end type safety.

## 🛠 Getting Started

To spin up the U-ORBIT platform locally:

### Prerequisites

Ensure you have [Node.js](https://nodejs.org/) (v18+ recommended) and a package manager (`npm`, `yarn`, `pnpm`, or `bun`) installed.

### Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/your-org/u-orbit.git
   cd u-orbit
   ```

2. **Install dependencies:**
   ```bash
   npm install
   # or yarn / pnpm / bun install
   ```

3. **Start the development server:**
   ```bash
   npm run dev
   ```

4. **Explore the Universe:**
   Open [http://localhost:3000](http://localhost:3000) in your browser. The page will auto-update as you edit files in the `app/` and `components/` directories.
